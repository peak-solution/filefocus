## 🟦 Program and Analyze

Your file based data is exposed via the [ASAM ODS](https://www.asam.net/standards/detail/ods/wiki/) standard HTTP-API and can be accessed using standard ASAM ODS tools - but also Python.  

In our *Jupyter Notebook* examples we are using the [ASAM ODSBox](https://peak-solution.github.io/odsbox/) - an open-source Python wrapper which hides away ASAM ODS complexities.

You can find examples for
* working with ASAM ODS data
* developing [ExD DataPlugins](dataingest.md#exd-dataplugins)

⚠️ Note: The example notebooks accessing the server via the internal web address of the docker network. If you're using notebooks outside the *PeakTDM FileFocus* examples you have to use the external web address, user and password. The current external web address of the server is shown on the bottom of the *PeakTDM FileFocus* app.

You can find more examples for *PeakTDM FileFocus*  in the [Data Management Learning Path](https://peak-solution.github.io/data_management_learning_path/).
